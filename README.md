# JAVA
### Course ID -- BSCS2005
### Course Credits -- 4
### Course Type -- Programming 

# Reference Book --
- Java The Complete Reference Eleventh Edition Paperback by Herbert Schildt
- Concepts in Programming Languages by John C. Mitchell


# Course hierarchy
- **Week 1** -- 
    Introduction to Programming, Types, Memory management, Abstraction and modularity, Object Oriented Programming, classes and Objects

- **Week 2** -- 
    Introduction to java, Basic data types in java, Control flow in java, Defining classes and objects in java

- **Week 3** --
    Object oriented programming, Subclasses and Inheritance, Dynamic dispatch and polymorphism, Class Hierarchy

- **Week 4** -- 
    Abstract classes, Interfaces, Private classes, Controlled interaction with objects, Callbacks, Iterators

- **Week 5** --
    Polymorphism, Generic programming, Java Generics and Subtyping, Reflection

- **Week 6** -- 
    Indirection, Collection, MAP

- **Week 7** --
    Errors and Exceptions, Packages, Assertions, Logging

- **Week 8** -- 
    Cloning, Type interface, Higher order functions, Streams

- **Week 9** -- 
    Optional types, Collecting Results from Streams, Input output streams, Serialisation

- **Week 10** --
    Concurrency:Threads and processes, Race conditions, Mutual Exclusion, Test and Set, Monitors 

- **Week 11** --
    Monitoring, Thread, Concurrent Programming, Thread safe collection

- **Week 12** --
    Graphical interface and event driven programming, Swing toolkit
